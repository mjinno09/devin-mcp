# Task Plans

タスクに着手する前に計画を書く。完了後はそのまま作業ログとして残す。

## フォーマット

ファイル名: `NNNN-<slug>.md`（例: `0001-initial-implementation.md`）

```
## NNNN. タイトル

### Goal
何を達成したいか。

### Background
このタスクの背景・動機。

### Approach
実装方針。ステップに分解する。
1. ...
2. ...
3. ...

### Out of Scope
今回やらないこと。

### Tasks
- [ ] タスク1
- [ ] タスク2
- [ ] タスク3

### Notes
作業中に気づいたこと、変更した判断などを追記する。
```
